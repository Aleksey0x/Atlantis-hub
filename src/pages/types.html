Выбор автономной онлайн-кассы

Быстрый разбор: для каких случаев подходит автономная онлайн-касса.

Atlantis
• 21 мая 2025

Основные типы онлайн-касс

Существует 4 основных типа онлайн-касс:

1.1 Автономные кассы

Работают как самостоятельные устройства: чековый принтер, дисплей и цифровая клавиатура — всё в одном корпусе.

  • Не требуют компьютера
  • Подходят для киосков, курьеров, уличной торговли
  • Просты в использовании

1.2 Смарт-терминалы

Похожи на смартфон: сенсорный дисплей, компактность, собственные приложения.

  • Встроенный принтер чеков
  • Передача данных в налоговую через интернет
  • Отличный выбор для небольших торговых точек

1.3 Фискальные регистраторы

Контрольно-кассовая техника с фискальным накопителем.

  • Работают только с кассовой программой на компьютере
  • Подходят для магазинов с большим ассортиментом
  • Высокая скорость печати и надежность

1.4 Облачные кассы

Программное решение для онлайн-продаж.

  • Используются в интернет-магазинах
  • Не печатают чеки физически — только электронные

Как выбрать онлайн-кассу: что важно учитывать

2.1 Технические особенности точки:

  • Тип подключения к интернету (Wi-Fi, 3G, Ethernet)
  • Наличие перебоев в связи
  • Есть ли розетки или нужна автономность

2.2 Поток клиентов:

  • Небольшие магазины: достаточно 75–100 мм/с скорости печати
  • Аптеки, супермаркеты, столовые: нужно от 200 мм/с + автоотрезчик
Фискальные регистраторы для супермаркетов должны выдерживать пиковую нагрузку: скорость не менее 200 мм/с, обязательно с автоотрезчиком.

2.3 Мобильность:

  • Для выездной торговли: компактность, аккумулятор, встроенный 3G/4G модуль
Для разъездной торговли подойдут модели с хорошим аккумулятором и встроенным модемом.

2.4 Способы подключения к интернету:

  • Ethernet
  • Wi-Fi
  • Встроенный 3G/GPRS-модем
  • USB через EoU

2.5 Работа с номенклатурой:

  • Автономные кассы — максимум 50 товаров, неудобный ввод
  • Смарт-терминалы и фискальные регистраторы — удобны при большом ассортименте

2.6 Простота в использовании:

  • Автономные кассы — самые понятные
  • Простое меню, сообщения об ошибках на экране

2.7 Стоимость:

  • Автономные кассы — самые дешёвые
  • Смарт-терминалы и регистраторы — дороже, но функциональнее
Модернизировать старую кассу до онлайн-версии чаще всего невыгодно. Лучше купить новую.

2.8 Режим фискального регистратора:

  • Некоторые автономки (например, АТОЛ 91Ф) умеют работать как регистратор, если подключить к ПК

2.9 Ширина чека:

  • 57 мм — стандарт
  • 80 мм — влезает больше информации, логотипы и пр.

2.10 Отрезчик:

  • Важно для точек с большим потоком
  • В бюджетных моделях — ручной отрыв через гребёнку

2.11 Габариты:

  • Маленькие кассы — для точек с ограниченным пространством

2.12 Интерфейсы:

  • Для подключения к ПК, денежному ящику, интернету и др.

2.13 Расходы на ПО:

  • Уточняйте наличие и стоимость лицензий, обновлений

2.14 Маркировка товаров:

  • Автономные кассы чаще всего не поддерживают
  • Смарт-терминалы и регистраторы — поддерживают

2.15 Продажа алкоголя (ЕГАИС):

  • Требуется товароучетная система
  • Автономки без доработки не подойдут
Без товароучетной программы продать алкоголь через онлайн-кассу нельзя.

2.16 Техподдержка:

  • Уточняйте режим работы, доступность удалённого подключения

2.17 Экономия на эквайринге:

  • Кассы с поддержкой СБП позволяют платить меньше комиссии

2.18 Интеграция с ПО и сайтами:

  • Позволяет автоматически передавать данные о продажах
  • Облегчает работу интернет-магазинам

Рекомендации по выбору

  • Выездная торговля / курьеры / киоски: автономные кассы или мобильные смарт-терминалы
  • Небольшие магазины с ассортиментом до 50 товаров: автономные кассы
  • Средний/высокий поток и большой ассортимент: смарт-терминалы или фискальные регистраторы + товароучёт
  • Продажа алкоголя / маркированных товаров: касса должна поддерживать ЕГАИС, маркировку и иметь совместимость с ПО на Windows
